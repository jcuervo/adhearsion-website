Adhearsion

Adhearsion is...

Adhearsion is a new way to write voice-enabled applications. Adhearsion is not just another API or library — it's a full-featured framework which streamlines the application building process and is optimized for what telephony application developers do most. Its name is a combination of "adhesion" and "hear" because Adhearsion shines best when adhering technologies together, especially with voice.

It's modern

Adhearsion uses the high-level Ruby programming language and boasts a comprehensive regression test suite. The so-called Web 2.0 era is about making the internet social, but voice has been an underutilized asset because of the traditional barriers to entry. Adhearsion aims to help merge these two areas.

It's open-source

We take open-source software seriously. The telephony industry, above all, needs fresh and innovative talent and Adhearsion's here to bring it.

It's fun

Voice-enabled applications are a relatively new concept to many people, especially developers in the modern web development space.
